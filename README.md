# SmCrypto

本项目是对[`sm-crypto`](https://github.com/JuneAndGreen/sm-crypto)（`JavaScript`实现）的其他语言实现，主要目的是为了同步其加密和解密接口。

计划中的实现语言，以及其目前各自的实现情况如下：

| 语言     | SM2算法 | SM3算法 | SM4算法      |
| -------- | ------- | ------- | ------------ |
| `Rust`   |         |         |              |
| `Python` |         |         | $\checkmark$ |
| `Go`     |         |         |              |
| `Nim`    |         |         |              |



## 使用示例

对于各语言的模块调用示例，请参考各语言的文件夹中的`README.md`。
